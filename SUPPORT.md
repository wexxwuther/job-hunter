# Support

A few ways to get help with `job-hunter`, in order of "fastest answer for you":

## Questions, ideas, and "did this work for you?"

Open a thread in [Discussions](https://github.com/wexxwuther/job-hunter/discussions). Other users see it; you get the answer; the thread becomes the answer for whoever asks next.

Good fits for Discussions:
- "Does this work with my harness?"
- "How do I tell the agent to skip a specific company?"
- "I used job-hunter to land a job, here's what worked"
- "What's the difference between `posted` and `applied_date`?"
- Anything that isn't broken but you wish was clearer

## Something is broken

Open an [Issue](https://github.com/wexxwuther/job-hunter/issues/new/choose). The bug-report template asks for the things that actually let me reproduce it. If you fill those in, this gets fixed faster than if you don't.

## You found a security problem

**Please don't open a public issue.** See [SECURITY.md](SECURITY.md) for the private-disclosure channel.

## You want to contribute

[CONTRIBUTING.md](CONTRIBUTING.md) has the workflow. PRs welcome. Bug fixes with a regression test are the easiest path to a merge.

## Response time

I work on this on the side. Expect a day or two for issues, sometimes a week. Discussions move faster because the community helps. Critical security reports get prioritized.

## What's NOT in scope

If you're looking for help with these, you'll get a faster answer elsewhere:
- General career advice (no posting in hand)
- Interview prep, out of scope for this skill by design
- Salary negotiation strategy, different trust model, different skill
- Auto-applying to jobs, never going to be a feature

For each of those, the agent harness you're using probably has a better-fit skill. Try a search on agentskills.io or your harness's skill directory.
